# ERROR CODES

0. -1 = Stop crrent event
1. 0 = Everything is ok
2. 1 = Auth error
3. 2 = Permission error
4. 3 = Verify error
5. 4 = Missing data
6. 5 = Already existing
7. 6 = Not found
8. 7 = Internal Server Error
